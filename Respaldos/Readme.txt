En esta carpeta se guardarán los respaldos generados, pero cada respaldo se descargará en la máquina donde se este realizando.

Para hacer la restuaración desde el sistema web se debe de utilizar el archivo descargado a la máquina donde fue creado el respaldo.

Para hacer la restauración desde el webmin (Directamente en el servidor) se puede utilizar el respaldo mas reciente que se encuentre en esta carpeta!


			IMPORTANTISIMO:	
	*Al archivo que se descargue en la máquina local cuando se cree el respaldo, no se le debe de cambiar el nombre.
	 Esto debido a que para hacer la restauración desde el sistema web se busca el respaldo deseado dentro de la carpeta Respaldos/ y en esa carpeta tiene el nombre con el que fue creado!
